**Overview & Objectives**

This document is the complete systematic framework for rebuilding 1iQ.ai
into a world-class development intelligence platform website. It covers
every page, every section, every component, every content requirement,
and the precise execution sequence for making this site ready for
high-net-worth real estate developers, family offices, and private
equity real estate groups.

The framework is organized in 8 phases, executed in order. Each phase
has specific deliverables, Antigravity prompts, acceptance criteria, and
dependency notes. No phase should begin until the prior phase is
validated.

+-----------------------------------------------------------------------+
| **North Star**                                                        |
|                                                                       |
| Every word, every design decision, every feature on this website must |
| pass one test: would a principal of a \$50M real estate development   |
| firm find this credible, relevant, and compelling enough to book a    |
| call? If not, it does not belong on the site.                         |
+-----------------------------------------------------------------------+

**Target Audience Profile**

Before building anything, the entire team must have a shared
understanding of exactly who this website is speaking to. Every copy
decision, every design choice, and every feature flows from this
profile.
